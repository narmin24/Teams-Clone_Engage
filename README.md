2. ReactJS
3. WebRTC
4. Socket.io
5. PeerJS
6. CSS
7. HTML



## 4. Usage
Clone this repository on your local device. Install NodeJS and set up the environment.
npm i express socket.io nodemon
npm run start

cd frontend
yarn start
